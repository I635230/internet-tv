# 手順書

本ドキュメントは、DB作成およびサンプルデータ挿入の手順書です。

削除方法も併せて記載しているので、間違えた場合は、削除してやり直してください。

## 1. データベース構築

### 1.1. Query

以下のQueryでデータベースを作成します。

```sql
CREATE DATABASE net_tv1;
USE net_tv1
```

データベース構築の工程は以上です。

正常にデータベース構築できたら、2. テーブル構築に進んでください。

### 1.2. データベース削除

```sql
DROP DATABASE net_tv1;
```

## 2. テーブル構築

テーブル構築では、外部キー制約があるので、順番が違うとエラーが出力されます。順番通りにQueryを実行してください。

また、順番を間違えた場合は、2.2 テーブル削除を参照してください。

### 2.1. Query

```sql
CREATE TABLE time_zones
(id INT PRIMARY KEY AUTO_INCREMENT, start TIME, end TIME);
```

```sql
CREATE TABLE categories
(category VARCHAR(30) PRIMARY KEY);
```

```sql
CREATE TABLE channels
(channel VARCHAR(30) PRIMARY KEY);
```

```sql
CREATE TABLE tv_shows
(id INT PRIMARY KEY AUTO_INCREMENT, 
title VARCHAR(50), detail TEXT, 
INDEX title_index (title));
```

```sql
CREATE TABLE tv_show_categories
(id BIGINT PRIMARY KEY AUTO_INCREMENT, 
title VARCHAR(50), category VARCHAR(30), 
FOREIGN KEY title_foreign (title) REFERENCES tv_shows(title), 
FOREIGN KEY category_foreign (category) REFERENCES categories(category));
```

```sql
CREATE TABLE episodes
(id BIGINT PRIMARY KEY AUTO_INCREMENT, season INT, chapter INT, tv_show_id INT, 
title VARCHAR(50), detail TEXT, duration INT, release_date DATE, views BIGINT, 
FOREIGN KEY title_foreign (tv_show_id) REFERENCES tv_shows(id), 
UNIQUE episode_unique (chapter, season, tv_show_id));
```

```sql
CREATE TABLE programs
(id BIGINT PRIMARY KEY AUTO_INCREMENT, 
channel VARCHAR(30), date DATE, time_zone_id INT, episode_id BIGINT, 
FOREIGN KEY channel_foreign (channel) REFERENCES channels(channel), 
FOREIGN KEY time_zone_id_foreign (time_zone_id) REFERENCES time_zones(id), 
UNIQUE program_unique (channel, date, time_zone_id), 
FOREIGN KEY episode_foreign (episode_id) REFERENCES episodes(id));
```

テーブル構築の工程は以上です。

正常にテーブル構築できたら、3. サンプルデータ挿入に進んでください。

### 2.2. テーブル削除

```sql
DROP TABLE programs;
DROP TABLE episodes;
DROP TABLE tv_show_categories;
DROP TABLE tv_shows;
DROP TABLE categories;
DROP TABLE channels;
DROP TABLE time_zones;
```
